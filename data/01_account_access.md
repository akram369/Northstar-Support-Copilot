# Account Access and Password Reset

## Standard password reset
From the Northstar sign-in page, select **Forgot password**, enter the verified email address, and use the link delivered within 10 minutes. Reset links expire after 30 minutes and can be used once. Check spam and corporate quarantine folders before requesting another link.

## Locked accounts
Five failed sign-in attempts within 15 minutes trigger a 30-minute lock. Password resets do not remove this lock. Wait for the lock window to end, then sign in with the newest password. A support agent may confirm lock status but cannot disclose or change authentication factors without identity verification.

## Multi-factor authentication
If the original authenticator is unavailable, use a saved recovery code. If no recovery code exists, an organization administrator must start an MFA recovery request. Support must escalate MFA recovery and must never ask a user to share passwords, one-time codes, or recovery codes.
